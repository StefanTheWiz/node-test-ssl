# node-test-ssl
Test SSL using a simple node.js server

## How do I get set up? ##

### 1. Change the `$DOMAIN` name in `start.sh` ###
### 2. Install package dependencie s ###
```
cd /path/to/node-test-ssl
npm install .
```
### 3. Run ###
```
npm start
```
### 4. Debug ###
```
npm install nodemon
npm run debug
```
